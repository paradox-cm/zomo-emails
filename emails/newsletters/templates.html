Template Documentation

Comprehensive guide to understanding, customizing, and implementing the ZOMO Health newsletter templates. Learn about the technical features, mobile responsiveness, and best practices for creating professional email campaigns.

Template Overview

The ZOMO Health newsletter template is a professional, responsive email template designed for partner communications. It features a modern design with light/dark mode support, mobile responsiveness, and Mailchimp compatibility.

  • 600px width: Optimized for email clients and mobile devices
  • One-column layout: Clean, focused design for maximum readability
  • Inline styles: Ensures consistent rendering across email clients
  • VML buttons: Outlook-compatible call-to-action buttons
  • Dark mode support: Automatic adaptation to user's system preferences
  • Mobile responsive: Fully responsive design with mobile-optimized typography and spacing

Template Structure

The template is organized into modular sections that can be easily customized or rearranged:

  1. Header: Logo and date display
  2. Headline: Main newsletter title with accent color
  3. Hero Image: Placeholder for featured image
  4. Greeting & Intro: Personalized opening message
  5. In This Issue: Table of contents with icons
  6. Content Sections: Modular blocks for different topics
  7. Team Spotlight: Staff introduction with avatar
  8. Client Highlight: Success story showcase
  9. Resources: Health news and links
  10. Contact Info: Partner representative details
  11. Social Bar: LinkedIn follow button
  12. Footer: Legal compliance and unsubscribe links

Customization Guide

To customize the template for different campaigns:

  • Content Updates: Replace placeholder text with campaign-specific content
  • Images: Update hero image and team photos with absolute URLs
  • Links: Replace example.com URLs with actual campaign links
  • Contact Info: Update partner representative details in the contact section
  • Preheader Text: Modify the hidden preheader for email previews
  • Subject Line: Update the title tag for email subject

Technical Features

  • Email Client Compatibility: Tested with Outlook, Gmail, Apple Mail, and major clients
  • Responsive Design: Adapts to mobile devices and different screen sizes
  • Mobile Optimization: Media queries for 600px and 480px breakpoints with responsive typography
  • Accessibility: Proper semantic HTML and alt text for images
  • Brand Consistency: Uses ZOMO Health color palette and typography
  • Performance: Optimized HTML structure for fast loading
  • Compliance: Includes required unsubscribe and preference links

Mobile Responsiveness

The template includes comprehensive mobile optimization for all device sizes:

  • Responsive Breakpoints: 600px and 480px media queries for optimal mobile experience
  • Adaptive Typography: Font sizes scale down appropriately on mobile devices
  • Mobile Spacing: Reduced padding and margins for better mobile fit
  • Responsive Images: Logo and hero image scale to fit mobile screens
  • Stacked Layout: Cards and elements stack vertically on mobile
  • Touch-Friendly: Optimized button sizes and spacing for touch interaction

Color Palette

The template uses ZOMO Health's official brand colors:

  • Light Mode: Background #FFFFFF, Text #0a1216, Primary #0d9488, Muted #64748b
  • Dark Mode: Background #0a1216, Text #f8fafc, Primary #2dd4bf, Muted #94a3b8
  • Automatic Switching: Colors adapt based on user's system preference
